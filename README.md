## Features
- Unlocks ALL cosmetics.
- Lightweight and non-invasive.
- Compatible with all Fabric versions of Lunar Client.

### Known Issues
- Cloth cloaks do not work.
- Cosmetic metadata is not saved.

### Future Plans
- Emotes.
- Staff modules.
- Restricted modules.
